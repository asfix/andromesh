About the Project - Targets

This project develops an innovative deep learning model that processes both byte images and function call graphs from Android APK files. Our approach combines the strengths of static image analysis with graph-based behavioral patterns to achieve superior malware detection accuracy.

Multi-Modal Architecture

Malnet-Medium Dataset

Our curated dataset extends the original Malnet with benign samples, containing 95,670 total samples across 294 malware families and benign applications. All samples are split into training (70%), validation (10%), and test (20%) sets. We offer DEX images their H5 Pytorch-ready graph counterparts for new academic studies. If you want to access the original APK files, you need to contact with us for this kind of exclusive use.
